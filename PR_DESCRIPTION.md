## 변경사항

### 1. 신고 기능 개선
- ✅ 로그인 확인 추가: 로그인하지 않은 사용자는 신고 불가
- ✅ 중복 신고 방지: Firebase의 `reportedBy` 배열과 로컬 모드의 Set을 사용하여 동일 사용자의 중복 신고 방지
- ✅ 동시성 처리: Firebase 모드에서 트랜잭션을 사용하여 여러 사용자가 동시에 신고할 때 데이터 일관성 보장
- ✅ 로컬 모드 지원: LocalAppRepository에 `reportProduct()` 및 `getReportedCount()` 메서드 추가
- ✅ 코드 품질 개선: 중복 코드 제거, 명확한 분기 처리, 적절한 에러 처리
- ✅ UI 개선: 불필요한 `setState()` 제거, SnackBar 텍스트 간결화

### 2. 관리자 페이지 기능 추가
- ✅ 탭 구조 추가: 광고 관리 / 신고된 상품 관리 탭 분리
- ✅ 신고된 상품 목록 조회: Firebase/로컬 모드 모두 지원
- ✅ 신고된 상품 처리 기능:
  - 상품 상세 페이지로 이동
  - 신고 해제 (신고 횟수 및 신고자 목록 초기화)
  - 상품 삭제 (확인 다이얼로그 포함)
- ✅ Pull-to-refresh 지원
- ✅ 신고 횟수 배지 및 판매자 정보 표시

### 3. 환영 화면 및 로그인 플로우 개선
- ✅ 첫 화면을 환영 화면으로 변경
- ✅ 환영 화면 UI 개선:
  - 로고 크기 확대 (80x80 → 100x100)
  - 그림자 효과 추가
  - 레이아웃 간격 조정
  - SafeArea 적용
- ✅ 카카오 로그인 제거: 이메일 로그인만 지원
- ✅ 환영 화면에서 AppBar 숨김 처리
- ✅ 로그인 성공 후 자동 화면 전환 추가

## 주요 개선사항

### 신고 기능
- **중복 신고 방지**: Firebase의 `reportedBy` 배열과 로컬 모드의 Set을 사용하여 동일 사용자의 중복 신고를 방지합니다.
- **동시성 처리**: Firebase 모드에서 트랜잭션을 사용하여 여러 사용자가 동시에 신고할 때 데이터 일관성을 보장합니다.
- **로컬 모드 지원**: LocalAppRepository에 신고 관련 메서드를 추가하여 로컬 모드에서도 신고 기능을 사용할 수 있습니다.

### 관리자 기능
- **신고된 상품 관리**: 관리자가 신고된 상품을 확인하고 처리할 수 있는 기능을 추가했습니다.
- **직관적인 UI**: 탭 구조로 광고 관리와 신고된 상품 관리를 분리하여 사용성을 개선했습니다.

### 사용자 경험 개선
- **환영 화면**: 앱 첫 실행 시 환영 메시지와 함께 깔끔한 UI를 제공합니다.
- **로그인 플로우**: 로그인 성공 후 자동으로 홈 화면으로 전환되어 사용자 경험이 개선되었습니다.
